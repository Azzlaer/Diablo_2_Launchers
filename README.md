Te vengo a mostrar el generador de Launcher con respecto a un servidor del juego
DIABLO II Lord of Destruction

El pack contiene un generador del archivo files.xml que el Launcher debe
buscar para poder descargar los archivos, asi mismo el Launcher se puede
modificar en casi todo aspecto para lograr mejorarlo, como este proyecto
nunca se uso procedo a liberarlo a la comunidad si es que alguien quiere 
hacer oficio para usarlo, dejo el proyecto abierto para su uso y/o comercializacion
del mismo.

Dejo tambien mis cuentas de PAYPAL y BINANCE por si desean donar y darme algun
regalo, lo dudo que llegue a pasar pero nunca es tarde para la persona que lo necesita.

Saludos
